# 統計的検証・アブレーション・外的妥当性レポート（2026-03-01）

## 実施内容
- 統計的検証: 建物ごとのペア差分に対して、bootstrap 95% CI + 符号反転 permutation 検定を実施（n=100/building）。
- アブレーション厳密化: `schema_only` / `material_only` / `self_refine_only` / `full_stack` を同一データ・同一評価で比較。
- 外的妥当性: `buildings_100_v1`（簡素）と `buildings_100_v4`（多様）で同条件を再評価し、分布ギャップを比較。

## 4. seed反復について
- 再標本化seed反復（seeds=`[11,22,33,44,55]`）を追加実行し、p値と差分推定の安定性を確認済み。
- 結果は `reports/statistics/seed_repeat_resampling/summary.md` と `summary.json` に保存。
- 主要比較（`full_vs_schema`, `full_vs_baseline`）で、IoU/F1 の差分符号は seed を跨いで安定。
- なお LLM生成そのものの seed 反復（description/plan 再生成）は別軸で、APIコストを伴うため次段実行を推奨。
- 実行基盤は整備済み: `--desc_llm_seed`, `--plan_llm_seed` を追加済み。
